# Interim Report

## Project Overview

This project focuses on insurance risk analytics and predictive modeling for AlphaCare Insurance Solutions (ACIS).

The objective is to:
- analyze historical insurance claims data,
- identify low-risk customer segments,
- validate statistical risk hypotheses,
- and develop predictive pricing models.

## Work Completed

### Task 1 — Exploratory Data Analysis
- Data cleaning and preprocessing
- Missing value assessment
- Loss ratio and margin analysis
- Geographic and temporal trend analysis
- Outlier detection
- Insight-driven visualizations

### Task 2 — Data Version Control (DVC)
- Initialized DVC pipeline
- Configured local remote storage
- Tracked raw and cleaned datasets
- Established reproducible workflow
